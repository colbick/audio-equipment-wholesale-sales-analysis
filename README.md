Wholesale Sales Data Analysis of Audio Equipment  
Анализ данных об оптовых продажах аудиотехники (EDA, аномалии, выручка, бренды, менеджеры)

Описание:
В проекте анализируются данные об оптовых заказах аудиотехники за месяц.  
Цели анализа — понять динамику заказов, выявить аномальные дни, разобрать влияние статусов (confirmed/canceled), рассчитать ключевые метрики в рублях с учетом курса USD и оценить бренды/ассортимент.

Инструменты, которые использовались в анализе:
 
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter Notebook

Данные
Используются три таблицы и файл с курсом валюты:

df_orders.csv (заказы)
- `order_id` — номер заказа  
- `product_id` — идентификатор товара  
- `quantity` — количество товара в заказе  
- `date` — дата заказа  
- `manager` — менеджер

df_order_status.csv (статусы + клиенты)
- `order_id` — номер заказа  
- `client_id` — идентификатор клиента  
- `status` — статус заказа (`confirmed` / `canceled`)

df_products.csv (товары)
- `id` — идентификатор товара  
- `name` — название товара (бренд, модель)  
- `price` — цена единицы товара (USD)

`usd_rate.txt (курс доллара)
Файл содержит курс USD на каждую дату. На его основе формируется таблица с колонками:
- `date`
- `currency_rate`



Шаги, проделанные в проекте:

1) Подготовка данных

2) Динамика заказов и сезонность

3) Анализ статусов заказов

4) Разбор аномального всплеска

5) Расчет ключевых метрик в рублях (с учетом курса USD)

6) Анализ брендов

7) Анализ ассортимента (“виртуальная полка”)
